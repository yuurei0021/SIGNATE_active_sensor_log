# Data Visualization Experiment

## 目的
センサーデータの特性を理解し、各動作クラス間のデータ分布の違いを可視化する。

## 可視化内容

### 1. 基本的なデータ可視化 (01_basic_visualization.png)
- クラス分布: running, walking, idle, stairsの件数
- サンプル時系列データ: 各クラスの代表的なX軸加速度パターン
- 加速度分布: X, Y, Z軸の全体的な分布（100サンプル）
- 加速度の大きさ分布: ベクトルの大きさのヒストグラム

### 2. クラス別平均値比較 (02_class_mean_comparison.png)
各クラス50サンプルに対して：
- X, Y, Z軸それぞれの平均加速度の箱ひげ図
- 加速度の大きさの平均値の箱ひげ図

### 3. クラス別標準偏差比較 (03_class_std_comparison.png)
各クラス50サンプルに対して：
- X, Y, Z軸それぞれの標準偏差の箱ひげ図
- 加速度の大きさの標準偏差の箱ひげ図

### 4. クラス別時系列パターン (04_class_time_series.png)
各クラスの最初の5サンプルについて：
- X, Y, Z軸の時系列データを重ねてプロット
- クラスごとの典型的なパターンを可視化

## 主な知見

### クラス間の特徴的な違い
- **running**: 標準偏差が大きく、周期的な変動が顕著
- **walking**: runningよりも変動が小さいが、周期性は保持
- **idle**: 標準偏差が最も小さく、ほぼ一定の値
- **stairs**: Y軸（垂直方向）の変動が特徴的、他のクラスと明確に区別可能

### モデリングへの示唆
- 平均値だけでなく、標準偏差や時系列パターンが重要な特徴量となる
- runningとwalkingは似た傾向を持つため、周波数成分の分析が有効かもしれない
- idleは明確に区別可能
- stairsは他のクラスと異なるY軸の特徴を持つため、軸ごとの特徴が重要
